jPhone SMS Desktop


Introduction:

jPhone SMS Desktop is a freeware software that allows the user to export the iPhone SMS(s) directly on your pc (in HTML format or TXT format), either using SFTP for jailbroken iphone or using the iTunes backup folder for jailbroken/not-jailbroken iPhone. Through this software, it’s also possible to view your text messages, once loaded, directly on your pc just double-clicking on the row that represents the message you want to read. The main feature is that this application it is written in Java. It means that it’s multi-platform so that you can run the application from Linux/Windows/MacOSX, the only thing you have to install is the JAVA6 JRE, not inferior version.


Requirements:

  • This is a multi-platform software, so you need an installation of JAVA6 JRE in you OS. You can download it from the Sun Microsystem site. Whit inferior version of JAVA JRE, the application won’t work.
  • Only if you want to load the messages via SFTP, you need a jailbroken iPhone where you have installed openSSH so that is available the access via SFTP.


Instructions:

Launching the software on Windows:
  • Just Double-click on launch.bat

Launching the software on Linux / Mac OS X:

  • Just right click on JPhoneSMSDesktop_v0_6b.jar and choose “Run with Java JVM” (or similar).
  • Another way is via Terminal. Just positionate yourself in the directory where there is the JPhoneSMSDesktop_v0_6b.jar file and just lanch the command: java -jar JPhoneSMSDesktopv1.jar
  • WARNING: This program only works with JAVA6. Check you version of java with the command: java -version


Usage:

First Step: Load SMS from iPhone. In this version you can choose to load your text messages via SFTP or using the iTunes Backup Directory.

Using SFTP:

  • Connect the pc and the iPhone to the same WLAN / LAN.
  • Just retrieve the ip of the iPhone and put it in the requested field.
  • The User name and the password are set yet because they are the default. If you have changed them just rewrite them in the requested fields.
  • Now what you have to do is click on the button “Load Sms From iPhone” and wait just a little bit.
  • Now the table is full of you messages and if you want to read one message just double click on the row that represents the message you want to read.

Using iTunes Backup Directory:

  • Click on “Load SMS from Backup” button.
  • Choose the iTunes backup directory of the iPhone backup frome where you want  to retrieve your text messages.
  • Wait a few seconds until the table is full of your messages.
  • If you want to read one message just double click on the row that represents the message you want to read.
  • In this version, the application works ONLY ON *.mddata files. I hope the next release will work on *.mdbackup.
  • On Vista the iTunes backup directory is under C:\Users\YOURUSERNAME\AppData\Roaming\Apple Computer\MobileSync\Backup\
  • On Windows Xp the iTunes backup directory is under C:\Documents and Settings\USERNAME\Application Data\Apple Computer\MobileSync\Backup
  • On MacOSX the iTunes backup directory is under USERNAME/Library/Application Support/Mobile Sync/Backup/

Second Step: Exporting Messages. Once you have loaded your messages, there are two ways to export your messages. The first is in TXT format, the second one is in the HTML format.

TXT format:

  • Click on “Export to TXT” button.
  • Choose the directory where you want to save your file .txt with you exported messages and click OK.
  • Now you have just created the file .txt of your exported messages. Just go to the directory where you saved it and take a look.

HTML format:

  • Click on “Export to HTML” button.
  • Choose the directory where you want to save your exported messages in html format and click OK.
  • If you choose to save your messages in this way, the result is a very static site that shows you some stats in the index.html file and shows your conversations as the iPhone app SMS shows you.
  • Go to the directory you have just created and open the index.html with your favorite browser.
